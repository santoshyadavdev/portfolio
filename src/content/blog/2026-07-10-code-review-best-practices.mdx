## How Code Review as Process Started

It was very common in otger industries to have an inspection process, just take an example of Cars you drive, you need to get them inspected every year to make sure you can drive it and its safe to drive it and nothing is broken.

The similar approach was adopted and championed by [Michael Fagan](https://en.wikipedia.org/wiki/Michael_Fagan_(software_designer)) which was named as [Fagan Inspection](https://en.wikipedia.org/wiki/Fagan_inspection)

Remember in old days, shipping software was not this easy, people used to ship floppy disk, later CDs, if you application has bug it means, shipping a new disk, every bug was expensive.

### How Fagan Inspection worked

The Fagan inspection proposed multiple steps, and at the end of which the code ie etiher merged or sent for more work.
